# 听宅人 · 数据复盘看板

小红书 & 抖音 账号数据复盘 dashboard（团队内部用）。

## 怎么打开

直接双击 `index.html` 用浏览器打开即可（需联网，图表库走 CDN）。

## 文件结构

```
social-dashboard/
├── index.html   ← 页面本体（结构 + 样式 + 渲染逻辑），一般不用动
├── data.js      ← 所有数据都在这里，更新只改这一个文件
└── README.md    ← 本说明
```

## 怎么更新数据

把创作者中心的最新截图发给我，我读出数字后更新 `data.js` 即可，页面自动刷新。

也可以试用本地一键更新器：

```
tools/auto-refresh/update-dashboard.command
```

这个工具会打开本机浏览器采集创作者中心接口数据，当前试运行版只输出脱敏后的采集结果，不会自动覆盖线上看板。

数据字段说明（写在 `data.js` 顶部注释里）：
- `value` 真实数值，来自截图
- `delta` 环比；截图没有就填 `null`，卡片不显示环比
- 标 `estimated:true` 的趋势折线 / 观看时段，数值是按截图**形状估算**的近似值，
  页面上会标注「形态依据截图估算」。拿到精确数字后替换即可。

### 复盘方法层在哪改

`data.js` 末尾的 `playbook`：
- `northStar.items` —— 北极星指标卡的内容（值 / 同类对比）
- `benchmarks` —— 动作判定阈值（`hitPlays` 爆款线、`goodCtr` 达标点击率、`lowCtr`/`lowPlays` 砍线），可随你的标准调。
